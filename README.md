# Demo Documentation Repository

Демонстрационный репозиторий для хранения документации системного аналитика.

В составе репозитория:

- минимальный пример Python-кода без зависимостей;
- папка `docs/` с примерами аналитических артефактов;
- Markdown-документы с таблицами, диаграммами `mermaid`, `plantuml` и вставкой схемы для `draw.io`.

## Минимальный код

Файл `src/order_total.py` содержит очень простой пример функции расчета суммы заказа.

Пример запуска:

```bash
python3 src/order_total.py
```

## Структура документации

Документация разложена по нескольким каталогам:

- `docs/overview` - общий обзор решения;
- `docs/requirements` - требования и сценарии;
- `docs/processes` - бизнес-процессы и поток обработки;
- `docs/integration` - интеграционные материалы.

Этот репозиторий можно использовать как тестовый пример для демонстрации структуры базы знаний аналитика.
