# Состояния обработки

## Упрощённые состояния

| Состояние | Описание |
| --- | --- |
| Draft | Пользователь только формирует заказ |
| Calculated | Сумма успешно рассчитана |
| Shown | Результат отображён на витрине |

## Комментарий аналитика

Для демонстрационного репозитория достаточно описать только логические состояния, без хранения сущности заказа в БД и без статусов оплаты.
