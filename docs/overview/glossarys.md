## Термины

| Термин              | Определение                                          |
|---------------------|------------------------------------------------------|
| Заказ               | Набор товаров, который пользователь планирует купить |
| Позиция заказа      | Товар с указанным количеством                        |
| Итоговая сумма      | Стоимость товаров плюс доставка                      |
| Бесплатная доставка | Признак, который срабатывает при выполнении порога   |

## Сокращения

| Сокращение | Расшифровка                           |
|------------|---------------------------------------|
| UI         | Пользовательский интерфейс            |
| API        | Интерфейс программного взаимодействия |
| MVP        | Минимально жизнеспособный продукт     |

## Примечание

В реальном проекте глоссарий обычно ведётся отдельно и пополняется по мере уточнения терминологии между бизнесом, аналитиком и разработкой.
