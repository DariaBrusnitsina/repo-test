## Сценарий UC-01. Расчёт суммы заказа

### Предусловия

-  Пользователь выбрал товар.
-  Витрина знает цену и количество.

### Основной поток

1. Пользователь нажимает кнопку расчёта.
2. Витрина передаёт в сервис цену, количество и доставку.
3. Сервис возвращает итоговую сумму.
4. Витрина показывает результат пользователю.

### Альтернативный поток

Если сумма превышает порог бесплатной доставки, витрина показывает соответствующий признак.
