## Назначение

Демо-проект описывает упрощённый сервис расчёта суммы заказа в интернет-магазине. Система получает цену товара, количество и стоимость доставки, после чего возвращает итоговую сумму.ллззз

## Границы решения

| Область                     | Входит в решение | Не входит в решение |
|-----------------------------|------------------|---------------------|
| Расчёт суммы                | Да               |                     |
| Правила бесплатной доставки | Да               |                     |
| Оплата заказа               |                  | Да                  |
| Управление складом          |                  | Да                  |
| Отправка уведомлений        |                  | Да                  |

## Участники

| Роль           | Описание                   |
|----------------|----------------------------|
| Покупатель     | Оформляет заказ            |
| Витрина        | Передаёт входные параметры |
| Сервис расчёта | Считает итоговую сумму     |

## Контекстная диаграмма

```mermaid
flowchart LR
    Customer["Покупатель"] -->|"Оформляет заказ"| Storefront["Витрина"]
    Storefront -->|"Цена, количество, доставка"| PricingService["Сервис расчёта"]
    PricingService -->|"Итоговая сумма"| Storefront
    Storefront -->|"Сумма к оплате"| Customer
```

## Ключевое допущение

Демо-версия не хранит данные в базе и не вызывает внешние сервисы.
