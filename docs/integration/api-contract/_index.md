## Назначение

В реальном проекте расчёт мог бы вызываться через внутренний HTTP API. Ниже приведён пример чернового контракта.

## Запрос

```json
{
  "price": 799.0,
  "quantity": 3,
  "delivery_fee": 250.0
}
```

## Ответ

```json
{
  "total": 2647.0,
  "free_delivery": false
}
```

## Таблица полей

| Поле            | Тип     | Обязательность | Описание                    |
|-----------------|---------|----------------|-----------------------------|
| `price`         | number  | Да             | Цена одной единицы товара   |
| `quantity`      | integer | Да             | Количество товара           |
| `delivery_fee`  | number  | Нет            | Стоимость доставки          |
| `total`         | number  | Да             | Итоговая сумма              |
| `free_delivery` | boolean | Да             | Признак бесплатной доставки |
